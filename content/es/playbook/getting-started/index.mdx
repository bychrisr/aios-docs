# Comenzando

Todo lo que necesitas para empezar a usar Synkra AIOS de manera efectiva. Ya sea que quieras una configuración rápida o un recorrido completo, elige el camino que se ajuste a tu disponibilidad.

## Enlaces Rápidos

- **[Guía de Inicio Rápido](/playbook/getting-started/quick-start)** -- Empieza a funcionar en minutos con una configuración mínima
- **[Incorporación en 60 Minutos](/playbook/getting-started/onboarding-60min)** -- Recorrido completo estructurado que cubre instalación, agentes, flujos de trabajo y gates de calidad

## Prerequisitos

Antes de comenzar, asegúrate de tener:

- **Node.js** 18.0.0 o superior
- **npm** 6.0.0 o superior
- **Git** instalado y configurado
- **GitHub CLI** (`gh`) para flujos de trabajo de push y PR
- Un IDE de IA como **Claude Code** (recomendado)

## Instalación

```bash
# Crear un nuevo proyecto
npx @synkra/aios-core init my-project

# O instalar en un proyecto existente
cd existing-project
npx @synkra/aios-core install
```

## Qué Aprender Primero

1. **Nuevo en AIOS?** Comienza con la [Incorporación en 60 Minutos](/playbook/getting-started/onboarding-60min) para una introducción estructurada
