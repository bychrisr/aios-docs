# Primeros Pasos

Todo lo que necesitas para empezar a usar Synkra AIOS de manera efectiva. Elige el camino que se ajuste a tu disponibilidad.

## Enlaces Rápidos

- **[Guía de Inicio Rápido](/playbook/getting-started/quick-start)** — Empieza a funcionar en 5 pasos, en menos de 10 minutos
- **[Incorporación en 60 Minutos](/playbook/getting-started/onboarding-60min)** — Recorrido completo estructurado: instalación, los 12 agentes, flujo de trabajo SDC y gates de calidad

## Requisitos Previos

Antes de comenzar, asegúrate de tener:

- **Node.js** 20.0.0 o superior
- **npm** 8.0.0 o superior
- **Git** instalado y configurado
- **GitHub CLI** (`gh`) para flujos de trabajo de push y PR
- Un IDE de IA — **Claude Code** (recomendado), Cursor, o VS Code con extensiones de IA

## Instalación

```bash
# Crear un nuevo proyecto
npx @synkra/aios-core init my-project

# O instalar en un proyecto existente
cd existing-project
npx @synkra/aios-core install
```
