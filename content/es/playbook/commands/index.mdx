# Referencia de Comandos

Referencia completa de todos los comandos del meta-agente Synkra AIOS. AIOS proporciona mas de 60 comandos organizados en 11 categorias, cubriendo desde navegacion y gestion de stories hasta generacion de codigo y administracion del sistema.

## Como Funcionan los Comandos

Todos los comandos de AIOS usan el prefijo `*` (asterisco). Los comandos son sensibles al contexto y pueden comportarse de forma diferente dependiendo de cual agente esta activo. Algunos comandos son universales (disponibles para todos los agentes), mientras que otros son exclusivos de agentes especificos.

### Sintaxis de Comandos

```
*nombre-del-comando {param-requerido} [param-opcional] [--flag valor]
```

- Los comandos comienzan con `*` (asterisco)
- Los parametros en `{llaves}` son requeridos
- Los parametros en `[corchetes]` son opcionales
- Las flags comienzan con `--` y pueden aceptar valores
- Multiples flags pueden combinarse en un solo comando

### Atajos de Comandos

Los comandos comunes tienen atajos de una letra:

| Atajo | Comando Completo |
|-------|-----------------|
| `*h` | `*help` |
| `*s` | `*status` |
| `*la` | `*list-agents` |
| `*lt` | `*list-tasks` |
| `*lw` | `*list-workflows` |
| `*ca` | `*create-agent` |
| `*ct` | `*create-task` |
| `*cw` | `*create-workflow` |
| `*a` | `*analyze-framework` |
| `*i` | `*improve-code-quality` |

```
Draft → Ready → InProgress → InReview → Done
```

