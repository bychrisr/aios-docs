# Comandos Esenciales

Referencia rápida para los comandos AIOS más importantes.

## Comandos Universales

| Comando | Descripción |
|---------|-------------|
| `*help` | Muestra comandos disponibles para el agente activo |
| `*guide` | Muestra guía de uso completa |
| `*session-info` | Muestra detalles de la sesión actual |
| `*yolo` | Cambia modo de permisos (ask → auto → explore) |
| `*exit` | Sale del modo de agente actual |

## Gestión de Historias

| Comando | Agente | Descripción |
|---------|--------|-------------|
| `*draft` | @sm | Crea una nueva historia |
| `*validate-story-draft` | @po | Valida una historia (checklist de 10 puntos) |
| `*develop {story-id}` | @dev | Empieza a implementar una historia |
| `*qa-gate` | @qa | Ejecuta verificaciones de QA gate |
| `*qa-loop {story-id}` | @qa | Empieza ciclo iterativo de QA |

## Arquitectura

| Comando | Agente | Descripción |
|---------|--------|-------------|
| `*create-full-stack-architecture` | @architect | Arquitectura completa full-stack |
| `*create-front-end-architecture` | @architect | Arquitectura de frontend |
| `*create-plan` | @architect | Crea plan de implementación |
| `*analyze-project-structure` | @architect | Analiza proyecto para nuevas features |

## Gestión de Proyecto

| Comando | Agente | Descripción |
|---------|--------|-------------|
| `*create-epic` | @pm | Crea una nueva épica |
| `*execute-epic` | @pm | Ejecuta flujo de épica |
| `*research {topic}` | @analyst | Investigación profunda sobre un tema |

## DevOps

| Comando | Agente | Descripción |
|---------|--------|-------------|
| `*push` | @devops | Sube cambios al remoto |
