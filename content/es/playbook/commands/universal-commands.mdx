# Comandos Universales

Estos comandos están disponibles en todos los agentes de AIOS en cualquier momento. Proporcionan ayuda contextual, estado del sistema y control de sesión sin importar qué agente esté activo.

## Ayuda e Información

| Comando | Descripción |
|---------|-------------|
| `*help` | Mostrar todos los comandos disponibles para el agente actual |
| `*status` | Mostrar el contexto actual: agente activo, historia, estado de la tarea |
| `*guide` | Mostrar una guía de uso detallada para el agente actualmente activo |

### Ejemplo de Uso

```bash
# Ver qué comandos puede ejecutar @qa
@qa *help

# Comprobar en qué historia estás trabajando y su estado actual
*status

# Obtener un recorrido detallado de las capacidades de @sm
@sm *guide
```

---

## Control de Sesión

| Comando | Descripción |
|---------|-------------|
| `*exit` | Finalizar la sesión con el agente actual |
| `*yolo` | Cambiar el modo de permiso: `ask` (confirmar cada acción), `auto` (sin confirmación), `explore` (modo de descubrimiento) |
| `*chat-mode` | Iniciar modo de conversación libre para lluvia de ideas y preguntas |

### Ejemplo de Uso

```bash
# Permitir que Dex ejecute sin pedir confirmación en cada paso
@dev *yolo

# Finalizar la sesión con el agente actual
*exit
```

---

## Consejos

- Dirígete a un agente antes de ejecutar sus comandos: `@dev *help` activa a Dex y muestra sus comandos
- `*status` es la forma más rápida de reorientarte después de cambiar de agente
- `*guide` da más contexto que `*help` — úsalo cuando estés aprendiendo las capacidades de un nuevo agente
