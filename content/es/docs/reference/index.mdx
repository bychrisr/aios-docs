# Referencia

Referencia de API, esquemas de configuración y especificaciones técnicas.

## Configuración

- **Configuración Central** — `.aios-core/core-config.yaml`
- **Configuración del Proyecto** — `.aios/config.yaml`
- **Definiciones de Agentes** — `.aios-core/agents/`
- **Definiciones de Tareas** — `.aios-core/development/tasks/`

## Esquemas

AIOS utiliza esquemas YAML para validar:
- Definiciones de personas de agentes
- Definiciones de flujos de trabajo de tareas
- Formato de archivo de historia
- Planes de ejecución de épicas

> La documentación de referencia se sincroniza desde `aios-core/`.
