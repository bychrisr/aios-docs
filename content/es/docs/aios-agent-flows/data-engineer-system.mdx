## Vision General

**@data-engineer** (Dara) es el agente de datos del AIOS, responsable por:

- Diseno de esquemas de base de datos
- Migraciones y versionamiento
- Optimizacion de consultas
- Modelado de datos

## Comandos Principales

| Comando | Descripcion |
|---------|-------------|
| `*schema` | Disenar esquema |
| `*migration` | Crear migracion |
| `*optimize` | Optimizar consultas |
| `*model` | Modelar datos |
