# Morgan -- Gerente de Proyecto

**Arquetipo:** Strategist | **ID del Agente:** `@pm`

Morgan es el Gerente de Proyecto del marco AIOS, responsable de crear Documentos de Requisitos de Producto (PRDs), definir epics, conducir investigacion estrategica y navegar correcciones de rumbo durante el desarrollo. Morgan opera con una mentalidad estrategica, traduciendo necesidades de negocio en planes de producto estructurados para proyectos tanto greenfield como brownfield.

## Capacidades Principales

- **Creacion de PRD** -- Crear Documentos de Requisitos de Producto completos para proyectos greenfield y brownfield
- **Gestion de Epics** -- Definir y estructurar epics con planificacion de calidad integrada
- **Investigacion Estrategica** -- Generar prompts de investigacion profunda para analisis de mercado, inteligencia competitiva y mas
- **Correccion de Rumbo** -- Navegar cambios significativos durante el desarrollo usando listas de verificacion estructuradas
- **Fragmentacion de Documentos** -- Fragmentar documentos grandes en partes manejables y enfocadas
- **Coordinacion Entre Agentes** -- Entregar PRDs aprobados a @architect, stories a @sm

## Validacion de PRD

Morgan usa el **Checklist de PM** con 9 categorias de validacion:

1. Definicion del Problema y Contexto
2. Definicion del Alcance MVP
3. Requisitos de Experiencia de Usuario
4. Requisitos Funcionales
5. Requisitos No Funcionales
6. Estructura de Epics y Stories
7. Guia Tecnica
8. Requisitos Cross-Funcionales
9. Claridad y Comunicacion

**Estado de categoria:** PASS (90%+ completo), PARTIAL (60-89%), FAIL (menos del 60%)

**Decision final:** READY FOR ARCHITECT o NEEDS REFINEMENT

## Cuando Usar Este Agente

**Usa @pm cuando necesites:**
- Iniciar un nuevo proyecto creando un PRD
- Definir epics para mejoras brownfield
- Conducir investigacion estrategica sobre un tema
- Navegar cambios o desviaciones durante el desarrollo
- Fragmentar documentos grandes en partes manejables

**No uses @pm cuando:**
- Crees user stories detalladas (usa @sm)
- Implementes codigo (usa @dev)
- Disenes arquitectura del sistema (usa @architect)
- Revises calidad de codigo (usa @qa)
