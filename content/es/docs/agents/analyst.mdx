# Sage el Analista

**ID del Agente:** `@analyst` | **Arquetipo:** Decoder | **Signo:** Pisces

## Descripcion General

Sage (Atlas) es el **Analista de Negocios** del marco AIOS. Este agente se especializa en conducir investigacion de mercado, analisis competitivo, sesiones de brainstorming estructurado y generacion de prompts de investigacion profunda. Sage transforma informacion cruda en insights accionables que impulsan decisiones de producto.

Operando con un enfoque curioso y basado en evidencia, Sage facilita fases de descubrimiento creando briefs de proyecto, realizando estimaciones de mercado (TAM/SAM/SOM), analizando competidores y ejecutando sesiones de ideacion estructuradas. Todas las salidas estan disenadas para ser directamente consumibles por agentes posteriores -- particularmente @pm para la creacion de PRDs.

Sage es el punto de partida para proyectos greenfield, donde una investigacion y descubrimiento exhaustivos sientan las bases para todo lo que sigue. En escenarios brownfield, Sage contribuye reportes de awareness ejecutivo que resumen hallazgos de deuda tecnica para stakeholders.

## Capacidades Principales

- **Investigacion de mercado** con calculos TAM/SAM/SOM y analisis de tendencias
- **Analisis competitivo** usando marcos como las Cinco Fuerzas de Porter y SWOT
- **Sesiones de brainstorming estructurado** con multiples tecnicas de ideacion
- **Generacion de prompts de investigacion profunda** para investigacion enfocada
- **Creacion de briefs de proyecto** resumiendo problema, solucion y alcance MVP
- **Sesiones de elicitacion avanzada** para refinamiento de requisitos
- **Reportes ejecutivos** para awareness de descubrimiento brownfield
- **Investigacion de usuario y cliente** incluyendo personas y Jobs-to-be-Done

## Participacion en Workflows

### Workflow Greenfield Fullstack (Fase 1)
Sage lidera la fase de **Descubrimiento y Planificacion**, creando el `project-brief.md` que alimenta la creacion del PRD de @pm. Esto incluye investigacion de mercado, analisis del panorama competitivo y alcance inicial del proyecto.

### Brownfield Discovery (Fase 9)
Sage produce el **Reporte de Awareness Ejecutivo** (`TECHNICAL-DEBT-REPORT.md`), traduciendo hallazgos de deuda tecnica de fases anteriores a lenguaje amigable para negocios para comunicacion con stakeholders.

### Spec Pipeline
Sage participa en la **Fase 3 (Research)** cuando la clase de complejidad es STANDARD o COMPLEX, conduciendo investigacion de dependencias que informa la especificacion.

## Autoridad y Limites

**Autoridad exclusiva sobre:**
- Investigacion de mercado y analisis competitivo
- Facilitacion de sesiones de brainstorming
- Creacion de briefs de proyecto
- Generacion de prompts de investigacion

**Entrega a:**
- **@pm (Morgan)** -- Investigacion, analisis y briefs de proyecto para creacion de PRD
- **@po (Pax)** -- Insights de mercado e inteligencia competitiva
- **@architect (Aria)** -- Hallazgos de investigacion tecnologica
- **@ux-design-expert (Pixel)** -- Investigacion de usuarios para decisiones de UX

**No maneja:**
- Arquitectura de sistemas (delegar a @architect)
- Implementacion de codigo (delegar a @dev)
- Operaciones de Git (delegar a @devops)

## Cuando Usar

Activa @analyst cuando necesites:
- Investigar un mercado antes de construir un producto
- Analizar competidores e identificar oportunidades de diferenciacion
- Ejecutar una sesion de brainstorming para explorar ideas
- Crear un brief de proyecto para una nueva iniciativa
- Generar prompts de investigacion profunda para investigacion enfocada
- Producir reportes ejecutivos sobre hallazgos tecnicos

**No** uses @analyst para diseno de arquitectura (usa @architect), implementacion de codigo (usa @dev), o gestion de stories (usa @sm o @po).
