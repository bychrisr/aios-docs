# Agentes

El marco Synkra AIOS orquesta 12 agentes IA especializados, cada uno con responsabilidades, comandos y patrones de colaboración distintos.

## Resumen de Agentes

| Agente | Nombre | Rol | Comandos Clave |
|--------|--------|-----|----------------|
| @dev | Dex | Desarrollador | `*develop`, `*implement` |
| @architect | Aria | Arquitecto del Sistema | `*create-full-stack-architecture`, `*create-plan` |
| @pm | Morgan | Gerente de Proyecto | `*create-epic`, `*execute-epic` |
| @po | Pax | Propietario del Producto | `*validate-story-draft` |
| @sm | River | Maestro Scrum | `*draft`, `*create-story` |
| @qa | Quinn | Aseguramiento de Calidad | `*qa-gate`, `*qa-loop` |
| @analyst | Atlas | Analista de Negocios | `*research`, `*analyze` |
| @devops | Gage | Ingeniero DevOps | `*push`, `*deploy` |
| @data-engineer | Dara | Ingeniero de Datos | `*design-schema` |
| @ux-design-expert | Uma | Diseñador UX | `*create-front-end-architecture` |
| @aios-master | Orion | Gobernador del Marco | `*help`, any task |
| @squad-creator | Nova | Creator | Creación, validación y gestión de squads |

## Activación

Activa cualquier agente mencionándolos: `@agent-name`

Cada agente sigue su persona definida, comandos y límites de colaboración. Consulta la documentación individual del agente para más detalles.

> Las definiciones de agentes se sincronizan desde `aios-core/agents/`.
