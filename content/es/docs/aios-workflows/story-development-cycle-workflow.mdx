## Vision General

Workflow para el ciclo completo de desarrollo de una story, incluyendo:

- Analisis de requisitos
- Implementacion
- Tests y validacion
- Code review
- Deploy

## Cuando Usar

- Cada story de desarrollo
- Despues de workflows greenfield/brownfield
- Ciclo iterativo de desarrollo
