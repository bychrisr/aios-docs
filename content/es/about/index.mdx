# Acerca de Synkra AIOS

**Synkra AIOS** (AI-Orchestrated System) es un sistema operativo impulsado por IA para el desarrollo de software. Proporciona una capa de orquestación inteligente que coordina 13 agentes de IA especializados para manejar flujos de trabajo de desarrollo complejos -- desde la recopilación de requisitos hasta el despliegue.

---

## ¿Qué es Synkra AIOS?

Synkra AIOS es un meta-framework que transforma la forma en que los equipos de software trabajan. En lugar de usar IA como un simple asistente de código, AIOS proporciona un sistema estructurado donde agentes especializados colaboran a través de flujos de trabajo definidos, gates de calidad y reglas de gobernanza.

En su núcleo, AIOS está construido alrededor del concepto de **meta-agente**: una IA que comprende su propia base de código y puede analizar, generar y mejorar código basándose en patrones aprendidos y convenciones del equipo.

---

## Características Principales

- **13 Agentes Especializados** -- Cada agente tiene un rol distinto: developer (@dev), architect (@architect), QA (@qa), project manager (@pm), product owner (@po), scrum master (@sm), DevOps (@devops), data engineer (@data-engineer), UX expert (@ux-design-expert), analyst (@analyst), squad creator (@squad-creator / Nova) y el gobernador del framework (@aios-master)
- **4 Flujos de Trabajo Principales** -- Story Development Cycle (SDC), QA Loop, Spec Pipeline y Brownfield Discovery
- **Gates de Calidad Automatizados** -- Validación de historia de 10 puntos, QA gate de 7 puntos e integración de auto-corrección con CodeRabbit
- **Template Engine V2** -- Plantillas estandarizadas para historias, épicas, documentos de arquitectura y definiciones de agentes
- **Gobernanza Constitucional** -- Reglas del framework aplicadas consistentemente en todos los agentes
- **Principios IDS** -- Sistema de Desarrollo Incremental: REUTILIZAR sobre ADAPTAR sobre CREAR
- **Capa de Memoria** -- Indexación semántica y gestión de contexto para comprensión inteligente del código
- **Squads Multi-Agente** -- Crea equipos personalizados de agentes para flujos de trabajo específicos del dominio

## ¿Para Quién Es?

| Audiencia | Cómo Ayuda AIOS |
|-----------|-----------------|
| **Desarrolladores individuales** | Proporciona flujos de trabajo estructurados, verificaciones de calidad automatizadas y revisión de código asistida por IA |
| **Equipos de desarrollo** | Coordina el trabajo a través de historias, sprints y gates de calidad con responsabilidades claras de agentes |
| **Equipos empresariales** | Ofrece gobernanza, trazabilidad y auditorías a través de reglas constitucionales y el framework IDS |
| **Tech leads y arquitectos** | Soporta diseño de sistemas, evaluación brownfield y toma de decisiones arquitectónicas |

## Cómo Funciona

1. **Instalar** -- Inicializa AIOS en tu proyecto con `npx @synkra/aios-core init`
2. **Activar agentes** -- Elige el agente para tu tarea actual (por ejemplo, `@dev` para desarrollo, `@pm` para gestión de proyectos)
3. **Ejecutar flujos de trabajo** -- Ejecuta flujos de trabajo estructurados como el Story Development Cycle o el Spec Pipeline
4. **Gates de calidad** -- Checklists automatizados y revisión de código aseguran que cada artefacto cumpla los estándares de calidad
5. **Enviar** -- Delega operaciones de push y despliegue a @devops

El meta-agente aprende de tus patrones de desarrollo y mejora su asistencia con el tiempo.

---

## Principios Fundamentales

- **Especialización de Agentes** -- Cada agente tiene un rol y experiencia distintos con límites de autoridad claros
- **Desarrollo Impulsado por Historias** -- Todo el trabajo es trazable a través de historias vinculadas a épicas y PRDs
- **Calidad Primero** -- Gates de calidad automatizados en cada etapa del ciclo de vida de desarrollo
- **Desarrollo Incremental (IDS)** -- REUTILIZAR artefactos existentes antes de adaptar o crear nuevos
- **Gobernanza Constitucional** -- Las reglas del framework se aplican consistentemente, incluyendo la regla de "No Invención" (Artículo IV)
