## How It Works

1. Type `@agent-name` to activate the agent persona
2. The agent loads its configuration and greets you
3. Use `*command` syntax to execute agent-specific commands
4. Type `*exit` to deactivate the agent

## Common Commands

Every agent supports these base commands:

| Command | Description |
|---------|-------------|
| `*help` | Show all available commands |
| `*guide` | Show comprehensive usage guide |
| `*session-info` | Current session details |
| `*yolo` | Toggle permission mode |
| `*exit` | Exit agent mode |
