# Agents

The Synkra AIOS framework orchestrates 12 specialized AI agents, each with distinct responsibilities, commands, and collaboration patterns.

## Agent Overview

| Agent | Name | Role | Key Commands |
|-------|------|------|-------------|
| @dev | Dex | Developer | `*develop`, `*implement` |
| @architect | Aria | System Architect | `*create-full-stack-architecture`, `*create-plan` |
| @pm | Morgan | Project Manager | `*create-epic`, `*execute-epic` |
| @po | Pax | Product Owner | `*validate-story-draft` |
| @sm | River | Scrum Master | `*draft`, `*create-story` |
| @qa | — | Quality Assurance | `*qa-gate`, `*qa-loop` |
| @analyst | Atlas | Business Analyst | `*research`, `*analyze` |
| @devops | Gage | DevOps Engineer | `*push`, `*deploy` |
| @data-engineer | Dara | Data Engineer | `*design-schema` |
| @ux-design-expert | Uma | UX Designer | `*create-front-end-architecture` |
| @aios-master | — | Framework Governor | `*help`, any task |

## Activation

Activate any agent by mentioning them: `@agent-name`

Each agent follows its defined persona, commands, and collaboration boundaries. See individual agent documentation for details.

> Agent definitions are synced from `aios-core/agents/`.
