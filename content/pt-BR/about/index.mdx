# Sobre o Synkra AIOS

**Synkra AIOS** (AI-Orchestrated System) é um sistema operacional alimentado por IA para desenvolvimento de software. Ele fornece uma camada inteligente de orquestração que coordena 13 agentes de IA especializados para lidar com fluxos de trabalho complexos de desenvolvimento -- da coleta de requisitos ao deploy.

---

## O que é o Synkra AIOS?

O Synkra AIOS é um meta-framework que transforma a forma como equipes de software trabalham. Em vez de usar IA como um simples assistente de código, o AIOS fornece um sistema estruturado onde agentes especializados colaboram através de workflows definidos, quality gates e regras de governança.

No seu núcleo, o AIOS é construído em torno do conceito de **meta-agente**: uma IA que entende seu próprio código-fonte e pode analisar, gerar e melhorar código com base em padrões aprendidos e convenções da equipe.

---

## Funcionalidades Principais

- **13 Agentes Especializados** -- Cada agente tem um papel distinto: desenvolvedor (@dev), arquiteto (@architect), QA (@qa), gerente de projetos (@pm), product owner (@po), scrum master (@sm), DevOps (@devops), engenheiro de dados (@data-engineer), especialista em UX (@ux-design-expert), analista (@analyst), criador de squads (@squad-creator / Nova) e o governador do framework (@aios-master)
- **4 Workflows Primários** -- Story Development Cycle (SDC), QA Loop, Spec Pipeline e Brownfield Discovery
- **Quality Gates Automatizados** -- Validação de story em 10 pontos, QA gate de 7 pontos e integração com CodeRabbit self-healing
- **Template Engine V2** -- Modelos padronizados para stories, epics, documentos de arquitetura e definições de agentes
- **Governança Constitucional** -- Regras do framework aplicadas de forma consistente em todos os agentes
- **Princípios IDS** -- Sistema de Desenvolvimento Incremental: REUSE sobre ADAPT sobre CREATE
- **Camada de Memória** -- Indexação semântica e gerenciamento de contexto para compreensão inteligente de código
- **Squads Multi-Agente** -- Crie equipes personalizadas de agentes para workflows específicos de domínio

## Para Quem é?

| Público | Como o AIOS Ajuda |
|---------|-------------------|
| **Desenvolvedores solo** | Fornece workflows estruturados, verificações automatizadas de qualidade e code review assistido por IA |
| **Equipes de desenvolvimento** | Coordena o trabalho através de stories, sprints e quality gates com responsabilidades claras de agentes |
| **Equipes enterprise** | Oferece governança, rastreabilidade e trilhas de auditoria através de regras constitucionais e do framework IDS |
| **Tech leads e arquitetos** | Suporta design de sistemas, avaliação brownfield e tomada de decisões arquiteturais |

## Como Funciona

1. **Instale** -- Inicialize o AIOS no seu projeto com `npx @synkra/aios-core init`
2. **Ative agentes** -- Escolha o agente para sua tarefa atual (ex: `@dev` para desenvolvimento, `@pm` para gestão de projetos)
3. **Execute workflows** -- Rode workflows estruturados como o Story Development Cycle ou Spec Pipeline
4. **Quality gates** -- Checklists automatizados e code review garantem que todo artefato atenda os padrões de qualidade
5. **Entregue** -- Delegue operações de push e deploy para @devops

O meta-agente aprende com seus padrões de desenvolvimento e melhora sua assistência ao longo do tempo.

---

## Princípios Fundamentais

- **Especialização de Agentes** -- Cada agente tem um papel e expertise distintos com limites claros de autoridade
- **Desenvolvimento Orientado por Stories** -- Todo trabalho é rastreável através de stories vinculadas a epics e PRDs
- **Qualidade em Primeiro Lugar** -- Quality gates automatizados em cada etapa do ciclo de desenvolvimento
- **Desenvolvimento Incremental (IDS)** -- REUSE artefatos existentes antes de adaptar ou criar novos
- **Governança Constitucional** -- Regras do framework são aplicadas de forma consistente, incluindo a regra "Sem Invenção" (Artigo IV)
