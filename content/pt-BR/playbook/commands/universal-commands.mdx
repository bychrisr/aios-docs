# Comandos Universais

Estes comandos estão disponíveis em todos os agentes AIOS a qualquer momento. Eles fornecem ajuda contextual, status do sistema e controle de sessão independentemente de qual agente está ativo.

## Ajuda e Informações

| Comando | Descrição |
|---------|-----------|
| `*help` | Mostrar todos os comandos disponíveis para o agente atual |
| `*status` | Exibir contexto atual: agente ativo, story, estado da task |
| `*guide` | Mostrar guia de uso detalhado para o agente atualmente ativo |

### Exemplo de Uso

```bash
# Ver quais comandos @qa pode executar
@qa *help

# Verificar em qual story você está trabalhando e seu status atual
*status

# Obter um passo a passo detalhado das capacidades de @sm
@sm *guide
```

---

## Controle de Sessão

| Comando | Descrição |
|---------|-----------|
| `*exit` | Encerrar a sessão com o agente atual |
| `*yolo` | Alternar modo de permissão: `ask` (confirmar cada ação), `auto` (sem confirmação), `explore` (modo de descoberta) |
| `*chat-mode` | Iniciar modo de conversa livre para brainstorming e perguntas |

### Exemplo de Uso

```bash
# Permitir que Dex execute sem pedir confirmação a cada etapa
@dev *yolo

# Encerrar a sessão com o agente atual
*exit
```

---

## Dicas

- Enderece um agente antes de executar seus comandos: `@dev *help` ativa Dex e mostra seus comandos
- `*status` é a forma mais rápida de se reorientar após trocar de agente
- `*guide` fornece mais contexto do que `*help` — use-o ao aprender as capacidades de um novo agente
