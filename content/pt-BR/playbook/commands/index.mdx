# Comandos Essenciais

Referência rápida para os comandos mais importantes do AIOS.

## Comandos Universais

| Comando | Descrição |
|---------|-----------|
| `*help` | Mostrar comandos disponíveis para o agente ativo |
| `*guide` | Mostrar guia de uso completo |
| `*session-info` | Mostrar detalhes da sessão atual |
| `*yolo` | Alternar modo de permissão (ask → auto → explore) |
| `*exit` | Sair do modo do agente atual |

## Gestão de Stories

| Comando | Agente | Descrição |
|---------|--------|-----------|
| `*draft` | @sm | Criar uma nova story |
| `*validate-story-draft` | @po | Validar uma story (checklist de 10 pontos) |
| `*develop {story-id}` | @dev | Começar a implementar uma story |
| `*qa-gate` | @qa | Executar verificações do quality gate |
| `*qa-loop {story-id}` | @qa | Iniciar loop iterativo de QA |

## Arquitetura

| Comando | Agente | Descrição |
|---------|--------|-----------|
| `*create-full-stack-architecture` | @architect | Arquitetura completa do sistema |
| `*create-front-end-architecture` | @architect | Arquitetura do frontend |
| `*create-plan` | @architect | Criar plano de implementação |
| `*analyze-project-structure` | @architect | Analisar projeto para novas features |

## Gestão de Projetos

| Comando | Agente | Descrição |
|---------|--------|-----------|
| `*create-epic` | @pm | Criar um novo epic |
| `*execute-epic` | @pm | Executar fluxo do epic |
