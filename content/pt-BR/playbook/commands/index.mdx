# Referência de Comandos AIOS

Bem-vindo à referência de comandos do Synkra AIOS. Esta seção do playbook é o seu guia central para interagir com o framework através da linha de comando.

Os comandos são a forma mais direta e poderosa de instruir os agentes de IA, orquestrar workflows e gerenciar o ciclo de vida de desenvolvimento de software.

## Filosofia de Comandos

O AIOS adota uma abordagem de "orquestração explícita". Em vez de tentar adivinhar intenções complexas, o sistema expõe um conjunto rico de comandos que podem ser combinados para alcançar resultados precisos.

- **Comandos Universais:** Um conjunto de comandos essenciais disponíveis em qualquer contexto.
- **Comandos do Master Agent:** Ações de alto nível para gerenciamento do framework, orquestração de workflows e operações meta.
- **Comandos Especializados:** Ações específicas de domínio, delegadas a agentes especialistas como `@dev`, `@qa` e `@architect`.

## Como Navegar Nesta Seção

Para facilitar a consulta, dividimos a referência de comandos nas seguintes páginas:

- **[Comandos Universais](./universal-commands):** Comandos essenciais como `*help` e `*status`. Comece por aqui se você é novo no AIOS.
- **[Comandos do Master Agent](./master-agent-commands):** O painel de controle principal para orquestrar o AIOS.
- **[Comandos Especializados](./specialized-commands):** Uma visão geral de como os comandos são delegados a agentes especialistas.

Use o menu de navegação para explorar cada seção em detalhe.
