# Começando

Tudo o que você precisa para começar a usar o Synkra AIOS de forma eficaz. Seja para uma configuração rápida ou um passo a passo completo, escolha o caminho que se encaixa no seu tempo.

## Links Rápidos

- **[Guia de Início Rápido](/playbook/getting-started/quick-start)** -- Comece a usar em minutos com uma configuração mínima
- **[Onboarding em 60 Minutos](/playbook/getting-started/onboarding-60min)** -- Passo a passo estruturado completo cobrindo instalação, agentes, workflows e quality gates

## Pré-requisitos

Antes de começar, certifique-se de ter:

- **Node.js** 18.0.0 ou superior
- **npm** 6.0.0 ou superior
- **Git** instalado e configurado
- **GitHub CLI** (`gh`) para workflows de push e PR
- Uma IDE de IA como **Claude Code** (recomendado)

## Instalação

```bash
# Criar um novo projeto
npx @synkra/aios-core init meu-projeto

# Ou instalar em um projeto existente
cd projeto-existente
npx @synkra/aios-core install
```

## O que Aprender Primeiro

1. **Novo no AIOS?** Comece com o [Onboarding em 60 Minutos](/playbook/getting-started/onboarding-60min) para uma introdução estruturada
2. **Já familiarizado?** Vá direto ao [Guia de Início Rápido](/playbook/getting-started/quick-start) e comece a trabalhar imediatamente
3. **Procurando seu papel?** Confira as [Trilhas de Aprendizado](/playbook/trails) para caminhos específicos por papel (desenvolvedor, PM, QA, arquiteto)
