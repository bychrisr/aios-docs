# Primeiros Passos

Tudo o que você precisa para começar a usar o Synkra AIOS de forma eficaz. Escolha o caminho que se encaixa no seu tempo.

## Links Rápidos

- **[Guia de Início Rápido](/playbook/getting-started/quick-start)** — Comece a usar em 5 passos, em menos de 10 minutos
- **[Onboarding em 60 Minutos](/playbook/getting-started/onboarding-60min)** — Passo a passo estruturado completo: instalação, todos os 12 agentes, workflow SDC e quality gates

## Pré-requisitos

Antes de começar, certifique-se de ter:

- **Node.js** 20.0.0 ou superior
- **npm** 8.0.0 ou superior
- **Git** instalado e configurado
- **GitHub CLI** (`gh`) para workflows de push e PR
- Uma IDE de IA — **Claude Code** (recomendado), Cursor ou VS Code com extensões de IA

## Instalação

```bash
# Criar um novo projeto
npx @synkra/aios-core init meu-projeto

# Ou instalar em um projeto existente
cd projeto-existente
npx @synkra/aios-core install
```
