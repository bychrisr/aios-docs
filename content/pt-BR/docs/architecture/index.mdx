# Arquitetura

Documentação de arquitetura do sistema para o framework Synkra AIOS.

## Arquitetura Central

O AIOS é um meta-framework que opera como uma camada sobre seu ambiente de desenvolvimento. Ele oferece:

- **Orquestração de Agentes** — 12 agentes especializados coordenados por workflows definidos
- **Sistema de Tarefas** — Definições de tarefas atômicas e executáveis com pré/pós-condições
- **Desenvolvimento Orientado a Stories** — Todo trabalho flui por stories validadas
- **Gates de Qualidade** — Verificações de qualidade automatizadas e manuais em cada etapa

## Documentos de Arquitetura

Os documentos de arquitetura para projetos específicos são criados pelo agente `@architect` usando o comando `*create-full-stack-architecture`.

> As definições de arquitetura central são sincronizadas a partir de `aios-core/`.
