# Referência

Referência de API, schemas de configuração e especificações técnicas.

## Configuração

- **Configuração Central** — `.aios-core/core-config.yaml`
- **Configuração do Projeto** — `.aios/config.yaml`
- **Definições de Agentes** — `.aios-core/agents/`
- **Definições de Tarefas** — `.aios-core/development/tasks/`

## Schemas

O AIOS usa schemas YAML para validar:
- Definições de persona de agentes
- Definições de workflow de tarefas
- Formato de arquivo de story
- Planos de execução de epics

> A documentação de referência é sincronizada a partir de `aios-core/`.
