# Essential Commands

Quick reference for the most important AIOS commands.

## Universal Commands

| Command | Description |
|---------|-------------|
| `*help` | Show available commands for active agent |
| `*guide` | Show comprehensive usage guide |
| `*session-info` | Show current session details |
| `*yolo` | Toggle permission mode (ask → auto → explore) |
| `*exit` | Exit current agent mode |

## Story Management

| Command | Agent | Description |
|---------|-------|-------------|
| `*draft` | @sm | Create a new story |
| `*validate-story-draft` | @po | Validate a story (10-point checklist) |
| `*develop {story-id}` | @dev | Start implementing a story |
| `*qa-gate` | @qa | Run quality gate checks |
| `*qa-loop {story-id}` | @qa | Start iterative QA loop |
