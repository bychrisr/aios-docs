# Command Reference

Complete reference for all Synkra AIOS meta-agent commands. AIOS provides 60+ commands organized into 11 categories, covering everything from navigation and story management to code generation and system administration.

## How Commands Work

All AIOS commands use the `*` (asterisk) prefix. Commands are context-aware and may behave differently depending on which agent is active. Some commands are universal (available to all agents), while others are exclusive to specific agents.

### Command Syntax

```
*command-name {required-param} [optional-param] [--flag value]
```

- Commands start with `*` (asterisk)
- Parameters in `{curly braces}` are required
- Parameters in `[square brackets]` are optional
- Flags start with `--` and may accept values
- Multiple flags can be combined in a single command

### Command Shortcuts

Common commands have single-letter shortcuts:

| Shortcut | Full Command |
|----------|-------------|
| `*h` | `*help` |

## Command Chaining

Commands can be chained for complex operations:

```bash
# Sequential execution with &&
*analyze-framework && *suggest-improvements && *generate-report

# Data flow with pipes
*analyze-code src/ | *improve-code-quality | *generate-tests

# Independent execution with ;
*backup ; *update ; *doctor --fix
```
