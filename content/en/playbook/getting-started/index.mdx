## Prerequisites

Before starting, make sure you have:

- **Node.js** 20.0.0 or higher
- **npm** 8.0.0 or higher
- **Git** installed and configured
- **GitHub CLI** (`gh`) for push and PR workflows
- An AI IDE — **Claude Code** (recommended), Cursor, or VS Code with AI extensions

## Installation

```bash
# Create a new project
npx @synkra/aios-core init my-project

# Or install in an existing project
cd existing-project
npx @synkra/aios-core install
```
