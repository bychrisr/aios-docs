## Quick Start for Contribution

To set up the local development environment for this site, follow these steps:

```bash
# Install dependencies
npm install

# Development (search will not work)
npm run dev

# Production build (includes Pagefind indexing)
npm run build

# Serve production build locally (with search)
npm start
```

## Content Synchronization

This site automatically syncs content from the official [`SynkraAI/aios-core`](https://github.com/SynkraAI/aios-core) repository.

- **Schedule:** Daily at 6:00 UTC
- **Workflow:** `.github/workflows/sync-content.yml`

## How to Contribute

To suggest edits to documentation content (guides, architecture, etc.), please contribute directly to the [`aios-core` repository](https://github.com/SynkraAI/aios-core). Changes will be automatically synced with this site.

For site infrastructure improvements (layout, search, i18n, functionalities), open an *issue* or *Pull Request* in this repository.
