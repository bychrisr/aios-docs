## Activation

Activate any agent by mentioning them in your prompt: `@agent-name`

Each agent follows its defined persona, commands, and collaboration boundaries. Click on any agent name above to see detailed documentation including capabilities, commands, workflow participation, and usage guidelines.

## Agent Selection Guide

Not sure which agent to use? Here is a quick reference:

| You want to... | Use this agent |
|----------------|----------------|
| Start a new project (PRD) | @pm (Morgan) |
| Plan architecture | @architect (Aria) |
| Create user stories | @sm (River) |
| Validate stories | @po (Pax) |
| Implement code | @dev (Dex) |
| Review and test code | @qa (Quinn) |
| Push code / create PRs | @devops (Gage) |
| Design database schema | @data-engineer (Dara) |
| Design UI/UX | @ux-design-expert (Pixel) |
| Conduct research | @analyst (Sage) |
| Create or modify framework components | @aios-master (Orion) |
| Create agent teams | @squad-creator (Nova) |
| Orchestrate any task directly | @aios-master (Orion) |
