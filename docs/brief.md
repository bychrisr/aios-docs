# Project Brief: AIOS Docs

## Executive Summary

**Synkra AIOS Docs** e um site de documentacao publica construido com **Nextra 4** (Next.js App Router) que transforma os 940+ arquivos markdown do repositorio `SynkraAI/aios-core` em uma experiencia visual rica, navegavel e multi-idioma. O site serve como portal de conhecimento para desenvolvedores, times e stakeholders que desejam entender, adotar e utilizar o framework AIOS. Dark mode como padrao, com suporte a PT-BR, EN e ES.

## Problem Statement

A documentacao do Synkra AIOS e um dos ativos mais valiosos do framework — **940+ arquivos markdown** cobrindo agentes, workflows, tasks, templates, guias, arquitetura, stories e dados de referencia. Porem:

**Estado Atual:**
- A documentacao vive exclusivamente dentro do repositorio Git (`SynkraAI/aios-core`), acessivel apenas via navegacao de arquivos no GitHub ou clone local
- Nao existe navegacao estruturada, busca full-text, ou hierarquia visual
- Desenvolvedores perdem tempo navegando em arvores de diretorio para encontrar informacoes

**Impacto:**
- Barreira de entrada alta para novos usuarios do AIOS
- Conteudo multi-idioma (pt, en, es) ja existe mas nao e aproveitado
- Stakeholders nao-tecnicos nao tem acesso pratico a documentacao
- A riqueza do conteudo e desperdicada sem uma interface adequada

## Proposed Solution

Site de documentacao com **Nextra 4** que agrega e apresenta a documentacao do aios-core em formato web navegavel, com busca full-text, i18n, dark mode padrao, e secao exclusiva de **Playbook/Guia Pratico**.

**Arquitetura:**
```
[SynkraAI/aios-core]  ──(GitHub Action cron diario)──>  [bychrisr/aios-docs]
                                                              │
                                                         content/
                                                         ├── docs/        ← docs do aios-core (sync automatico)
                                                         ├── playbook/    ← guias praticos (conteudo proprio)
                                                         └── _meta.json   ← navegacao estruturada
                                                              │
                                                         Vercel (auto-deploy on push)
```

**Componentes-chave:**
1. **Documentacao Core** — 940+ markdown files do aios-core
2. **Playbook/Guia Pratico** — 19+ guias generalizados (greenfield, brownfield, retomada, crise, checklists, templates, comandos essenciais, trilhas por perfil, onboarding)
3. **Multi-idioma** — PT-BR, EN, ES
4. **Sync automatizado** — GitHub Action diario
5. **Search full-text** — Pagefind (nativo Nextra 4)
6. **Deploy Vercel** — Auto-deploy no push
