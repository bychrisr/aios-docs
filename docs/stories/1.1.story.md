## Description

Como desenvolvedor do AIOS, eu preciso de um projeto Nextra 4 funcional com Next.js 15 App Router para que a documentacao do framework possa ser renderizada como um site web navegavel.

Esta e a story fundacional — sem ela, nenhuma outra pode comecar. O objetivo e sair do zero para um site Nextra funcional com:
- Dependencias instaladas (nextra, nextra-theme-docs, next, react)
- Configuracao basica (next.config.mjs, app/layout.tsx, mdx-components.tsx)
- Content directory convention configurada
- Uma pagina placeholder renderizando
- Build passando com sucesso
- Dark mode habilitado como padrao (Nextra default)

## Acceptance Criteria

- [ ] **AC1:** `npm install` completa sem erros com nextra 4.x, nextra-theme-docs, next 15.x, react 19.x, typescript
- [ ] **AC2:** `npm run dev` inicia o dev server na porta 3000 sem erros
- [ ] **AC3:** `npm run build` completa com sucesso (SSG)
- [ ] **AC4:** Acessar `http://localhost:3000` mostra uma pagina com o titulo "Synkra AIOS Docs"
- [ ] **AC5:** A pagina renderiza em dark mode por padrao
- [ ] **AC6:** O arquivo `content/index.mdx` e renderizado como pagina principal via catch-all route
- [ ] **AC7:** O projeto usa TypeScript para configs e components
- [ ] **AC8:** `npm run lint` passa sem erros

## Scope
